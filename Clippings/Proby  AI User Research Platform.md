K-pop 사용이해 조사

### Key Findings

지인의 추천으로 시작해 아티스트의 내면과 소통에 깊이 빠져들게 됩니다.

| 아티스트의 실력과 콘셉트 | 무대 위에서 보여주는 화려한 퍼포먼스와 아티스트의 비주얼이 시각적인 충격을 주며 팬이 됩니다.  - • 노래와 춤 실력은 물론, 아티스트가 풍기는 독특한 분위기와 콘셉트에 매료됩니다. - • TV나 온라인 플랫폼을 통해 접한 무대 장악력이 본격적으로 빠져드는 계기가 됩니다. | 버추얼 아이돌의 기술적인 신선함이나 완성도 높은 음악 퀄리티가 기존과 다른 매력으로 다가옵니다.  - • 새로운 기술이 접목된 아이돌의 모습에 호기심을 느끼며 실력에 감탄하게 됩니다. - • 음악 자체의 세련미와 안무 실력을 중요하게 생각하며 깊이 몰입합니다. |  |  |  |
| 소통 경험과 활동의 기한 |  |  |  | SNS에서 아티스트에게 직접 답글을 받는 소중한 경험이 아티스트를 가족처럼 가깝게 느끼도록 만듭니다.  - • 내 댓글에 아티스트가 반응해주는 순간 팬으로서의 유대감이 급격히 상승합니다. - • 가사를 번역하고 영상을 공부하며 아티스트를 삶의 일부로 받아들입니다. | 그룹의 활동 기간이 정해져 있다는 소식을 접하면 지금 이 순간의 활동을 하나도 놓치지 않으려 더 깊게 몰입합니다.  - • 활동 기한이 한정적인 서바이벌 그룹의 특성상 현재의 콘텐츠 소비에 더 열성적입니다. - • 해체 소식이나 공백기를 앞두고 있을 때 아티스트를 지지하고 싶은 마음이 더 커집니다. |

아티스트와의 유대감을 바탕으로 상징성과 실용성을 모두 고려하여 소비합니다.
