# Proby 사용 핵심 이유 조사 — 설문 설계 (시니어 리서처 관점)

## 1. 리서치 목표

- **Primary**: Proby를 쓰는 **가장 핵심적인 이유**(1~3개)를 정량·정성으로 파악
- **Secondary**: 사용 목적·대안 경험·가치 인식 구조를 이해해 포지셔닝·메시징에 반영

---

## 2. 설계 원칙

- **짧게 유지**: 핵심 이유 집중 → 10문항 내외, 2~3분 완료 수준
- **행동·결과 우선**: "왜 쓰나요?"보다 "무엇을 하려고 / 무엇이 달라져서 쓰나요?"로 묻기
- **대안 맥락**: 이전에 썼던 방법·도구를 한 번만 묻어서 "대체 무엇을 대체한 것인지" 파악
- **가치 질문 1개**: "Proby가 없었다면?"으로 손실 가치( WTP·중요도) 간접 측정
- **마지막에 한 줄 개방형**: 정량 요약 후 "한 문장으로 핵심 이유" 수집

---

## 3. 문항 구조

| # | 목적 | 문항 요지 | 타입 |
|---|------|-----------|------|
| 1 | 스크리닝·맥락 | Proby 사용 빈도 | choice |
| 2 | 사용 목적(복수) | 주로 어떤 일에 Proby를 쓰나요 | checkbox |
| 3 | 핵심 목적(단일) | 그중 가장 중요한 사용 목적 하나 | choice |
| 4 | 도입 계기 | Proby 도입 전에 같은 일을 어떻게 했나요 | choice |
| 5 | 선택 이유(복수) | Proby를 선택·유지하는 이유 (복수) | checkbox |
| 6 | 가장 큰 이유(단일) | 위 중 가장 큰 이유 하나 | choice |
| 7 | 가치 인식 | Proby가 없었다면 그 일을 어떻게 했을 것 같나요 | choice |

## 4. 문항·선택지 상세 (실제 설문 문구)

1. **Proby를 얼마나 자주 사용하시나요?**  
   - 거의 매일 / 주 1~2회 / 월 1~2회 / 가끔 / 아직 시험만 해봄  

2. **주로 어떤 일에 Proby를 사용하시나요? (복수 선택)**  
   - 사용자 인터뷰·심층 조사 / 설문·정량 보완 / 내부 의견·피드백 수집 / 리서치 교육·연습 / 기타  

3. **그중에서 가장 중요한 사용 목적은 무엇인가요? (하나만)**  
   - (1번과 동일 선택지)  

4. **Proby 도입 전에는 같은 일을 주로 어떻게 하셨나요?**  
   - 직접 대면·화상 인터뷰 / 외주 리서치·에이전시 / 설문만 사용 / 거의 안 했음 / 기타  

5. **Proby를 선택하거나 계속 쓰는 이유는 무엇인가요? (복수 선택)**  
   - 속도·일정 단축 / 비용 절감 / 인력 부족 보완 / 인터뷰 품질·일관성 / 규모(다수 응답) 확보 / 글로벌·다국어 / 기타  

6. **위 중 가장 큰 이유는 무엇인가요? (하나만)**  
   - (5번과 동일 선택지)  

7. **Proby가 없었다면, 그 일을 어떻게 했을 것 같으신가요?**  
   - 다른 툴로 대체 / 외주·에이전시에 맡김 / 그만두거나 크게 줄임 / 잘 모르겠음  

8. **지금까지 Proby 사용 경험에 얼마나 만족하시나요?**  
   - 1(매우 불만) ~ 5(매우 만족) scale  

9. **주변 동료·지인에게 Proby를 추천할 의향이 있으신가요?**  
   - 1(전혀 없음) ~ 5(적극 추천) scale  

10. **Proby를 쓰는 가장 핵심적인 이유를 한 문장으로 적어주세요. (선택)**  
    - 단답이 아닌 짧은 문장 → paragraph  

11. **귀하의 주된 역할은 무엇에 가깝습니까?**  
    - 리서처 / PM·기획 / 마케팅·영업 / 경영·전략 / 기타  

12. **회사(또는 팀) 규모**  
    - 1인 / 2~10명 / 11~50명 / 51명 이상  

---

## 5. 분석 시 활용 포인트

- **핵심 이유**: 5·6번(복수+단일) + 10번(한 줄) 조합으로 요약 문구 도출
- **가치**: 7번으로 "없었으면 어떻게 했을지" → 대안 비용·중요도 해석
- **세그먼트**: 11·12번으로 역할·규모별 핵심 이유 차이 비교
